Le projet

Intent Analytics est une initiative d'exploration de données publiques françaises. L'objectif : montrer ce qu'on peut apprendre en croisant les informations publiquement disponibles, en les rendant accessibles et en documentant chaque étape de la chaîne de traitement.

Contexte

Les administrations françaises publient chaque année des millions de données d'excellente qualité :

  • DVF — Transactions immobilières notariées (DGFiP)
  • DPE — Diagnostics de performance énergétique (ADEME)
  • BPE — Base permanente des équipements (INSEE)
  • BAN — Base adresse nationale (IGN, La Poste, collectivités)
  • Données démographiques — INSEE IRIS, géographie

Ces données sont publiques, gratuites et conformes à une licence ouverte. Pourtant, elles restent fragmentées et difficiles à explorer pour la plupart des utilisateurs.

Motivation

Ce projet explore la question suivante : qu'est-ce qu'on peut apprendre en croisant ces données ?

  • Quels territoires devrait-on monitorer pour une veille immobilière ?
  • Quels patterns émergent des statistiques sur l'énergie ?
  • Comment visualiser efficacement des millions de points de données ?
  • Comment rendre reproductibles et transparentes les analyses statistiques ?

Intent Analytics propose des réponses expérimentales à ces questions, dans un esprit pédagogique autour de l'open data français.

Principes

1. Transparence — Chaque chiffre affiché est reproductible à partir des sources brutes et des formules documentées.

2. Confiance graduée — Les résultats affichent un score de confiance basé sur le volume de données, la fraîcheur et le nombre de sources croisées.

3. Accessibilité — Les explorations sont gratuites et accessibles sans inscription. Les données publiques le restent.

4. Open source pédagogique — La méthodologie est documentée pour que d'autres puissent la reproduire, l'améliorer ou la contester.

Structure

Le projet comprend :

  • Pipeline ETL — Récupération des données via APIs officielles, nettoyage, normalisation géographique (réconciliation INSEE)
  • Data Warehouse — Agrégations pré-calculées par commune (statistiques DVF, profils DPE, scores équipements)
  • Explorations web — Interfaces pour explorer les données, filtrer, comparer des communes
  • Documentation — Méthodologie détaillée, limites connues, sources et licences

État actuel

Le projet est en phase prototype. Les données couvrent actuellement le territoire français avec un focus sur les régions d'IDF et les grandes métropoles. Les explorations disponibles incluent l'analyse DVF et le scanner DPE. D'autres analyses (équipements, données démographiques) sont en cours de développement.

Limitations

Intent Analytics exploite des données publiques de bonne qualité, mais avec des limites :

  • DVF — Pas de données de SCI, donations, viagers. Délai de publication : environ 6 mois.
  • DPE — Couverture progressive depuis juillet 2021 (nouveau DPE). Répartition inégale par région.
  • BPE — Mise à jour annuelle. Couverture variable selon les communes.
  • Confiance — À faible volume (< 5 transactions/an), les indicateurs sont approximatifs et doivent être signalés à l'utilisateur.
