Données traitées

Quatre sources de données ouvertes, croisées et enrichies pour une vision complète du territoire.

DVF
12.2M transactions
Demandes de valeurs foncières — chaque vente immobilière en France depuis 2018.
DPE
9.8M diagnostics
Diagnostics de performance énergétique ADEME — classe énergie et émissions CO2.
BPE
236k équipements
Base permanente des équipements INSEE — commerces, santé, éducation, transports.
BAN
6.3M adresses
Base adresse nationale — géocodage précis de chaque adresse du territoire.
