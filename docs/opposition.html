Demande d'opposition

Cette page vous permet, en tant que propriétaire ou occupant d'un logement, de demander l'exclusion de votre bien des analyses publiques d'Intent Analytics. La demande prend effet immédiatement.

À qui s'adresse cette page ? Vous pouvez utiliser ce formulaire si vous êtes propriétaire ou occupant d'un logement et que vous souhaitez retirer celui-ci de nos analyses, indépendamment de toute utilisation antérieure de notre service. Si vous avez utilisé Intent Analytics et souhaitez gérer vos données personnelles, utilisez plutôt la page Mes données.

Identifier le logement à retirer

Saisissez l'adresse exacte du logement. Nous nous appuyons sur la Base Adresse Nationale officielle pour identifier le bien de manière non ambiguë.

Saisissez au moins le numéro et le nom de rue avec la commune ou le code postal.

Effets de l'opposition

L'inscription de votre adresse au registre d'opposition produit immédiatement les effets suivants : votre logement n'apparaît plus dans aucune analyse publique d'Intent Analytics, aucun nouveau rapport ne peut être généré sur cette adresse, et toute donnée passée correspondante est masquée des résultats.

Important : nous ne pouvons pas effacer rétroactivement les données déjà publiées par les sources publiques officielles (DVF, ADEME, BAN), qui restent disponibles sur les sites gouvernementaux d'origine.
