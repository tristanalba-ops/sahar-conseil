Fraîcheur et traçabilité

Chaque table source porte les métadonnées suivantes :

  • source_dataset : identifiant du jeu de données d'origine
  • imported_at : horodatage d'import initial
  • updated_at : horodatage de la dernière mise à jour
  • ban_match_score : indice de confiance du géocodage (0–100)
  • ban_match_method : méthode de matching (exact, high, good, medium, low)
Intent Analytics ne redistribue pas de données.

Les volumes affichés et les explorations dérivées servent uniquement à rendre la lecture publique plus intelligible. Les jeux sources restent accessibles chez leurs producteurs respectifs.
