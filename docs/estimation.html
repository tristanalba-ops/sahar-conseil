Explorer une commune

Saisissez un code INSEE ou un nom de commune pour obtenir un profil complet : prix, DPE, équipements, démographie.

Intent Analytics MCP

Connectez l'API Intent Analytics directement à Claude pour interroger les données en langage naturel. Posez vos questions sur n'importe quelle commune française.

https://intent-analytics-mcp-production.up.railway.app/sse
1

Ajouter le MCP

Dans les réglages de Claude Desktop ou Claude Code, ajoutez un MCP serveur avec l'URL SSE ci-dessus.

2

Poser des questions

Claude accède à 8 outils d'analyse : estimation, profil commune, recherche multicritères, aides rénovation, etc.

3

Exemples de prompts

"Quel est le prix au m² à Lyon 3e ?", "Quelles aides pour rénover un F en DPE G ?", "Communes abordables proches de Bordeaux ?"

Outils disponibles

estimate

Estimation immobilière à partir d'une adresse, surface et type de bien

enrich

Enrichissement complet d'une adresse : géocodage + données contextuelles

get_commune_profile

Profil complet d'une commune : prix, DPE, équipements, démographie

search

Recherche multicritères de communes par prix, DPE, équipements

search_nearby

Communes proches d'un point avec rayon configurable

get_renovation_aids

Aides à la rénovation énergétique selon le profil du bien

query

Requête SQL libre sur les données agrégées (lecture seule)

health

Vérification de l'état des services et fraîcheur des données
