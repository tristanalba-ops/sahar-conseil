Données sources

100% données officielles.

SAHAR exploite uniquement les sources publiques officielles françaises — gratuites, régulièrement mises à jour, vérifiables.

DVF — data.gouv.fr
Transactions notariées depuis 2019. Prix signés, adresse, surface, coordonnées GPS.
DPE — ADEME
10M+ diagnostics énergétiques. Étiquette, consommation, adresse, type.
INSEE IRIS / BPE
Population, revenus, équipements à l'échelle infra-communale.
SIRENE — INSEE
Toutes les entreprises françaises avec code NAF, effectif, localisation.
DARES
Tensions de recrutement par métier et bassin d'emploi. Mis à jour trimestriellement.
Cadastre Etalab
Parcelles cadastrales vectorielles. BAN pour géocodage des adresses.
