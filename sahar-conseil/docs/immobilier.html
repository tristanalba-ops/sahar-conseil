Ce que ça fait

Quatre fonctions. Un pipeline complet.

→ Analyse des transactions DVF

Filtrez par code postal, surface, prix, période. Visualisez les prix au m², identifiez les outliers, exportez en Excel.

→ Carte interactive

Chaque transaction affichée sur carte avec prix et score de surface. Identifier visuellement les zones sous-cotées.

→ Score sous-valorisation 0–100

Algorithme multicritères : prix au m² vs médiane locale, surface, type, date de transaction. Priorisez les contacts les plus pertinents.

→ Pipeline CRM intégré

De la détection au closing dans le même outil. Clic-to-call, SMS, WhatsApp, email. Dictaphone compte-rendu vocal.
